# Get started

## Configure developer portal profile

1. Go to https://appconnect.labs.ringcentral.com/console and create a new plugin
2. On setting page, switch to JSON viewer with the button on top-right corner
3. Under `src/manifests`, copy the json object over and save (recommend to start with `sync.json`)

## Local run

1. npm i
2. npm run dev

All set!
